# 効果音ファイルについて

このディレクトリには以下の効果音ファイルが必要です：

## 必要なファイル

1. **roulette-spin.mp3** (約5秒)
   - ルーレットが回転する音
   - 推奨: カジノのルーレット音、ホイール回転音

2. **victory.mp3** (2-3秒)
   - 勝利時の効果音
   - 推奨: ファンファーレ、勝利音、祝福音

3. **click.mp3** (0.5秒) *オプション*
   - ボタンクリック音
   - 推奨: カチッという短いクリック音

## 推奨フリー素材サイト

### 日本語サイト
- **効果音ラボ**: https://soundeffect-lab.info/
  - カテゴリ: システム音、ゲーム、演出・アニメ
- **DOVA-SYNDROME**: https://dova-s.jp/
  - 効果音カテゴリから検索

### 英語サイト
- **Freesound**: https://freesound.org/
- **Zapsplat**: https://www.zapsplat.com/
- **Mixkit**: https://mixkit.co/free-sound-effects/

## ファイル形式
- 形式: MP3
- ビットレート: 64-128kbps（軽量化のため）
- サンプリングレート: 44.1kHz

## ダウンロード後
1. ファイル名を上記の通りにリネーム
2. このディレクトリ (`public/sounds/`) に配置
3. アプリケーションを再起動

## 注意事項
- 必ず利用規約を確認してください
- 商用利用可能なものを選択してください
- クレジット表記が必要な場合は適切に表記してください
